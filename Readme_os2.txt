Tile World 2
Version 2.1.0
2026-04-25
-------------------
Tile World is an emulation of the game "Chip's Challenge" for the Atari
Lynx, created by Chuck Sommerville, and later ported to MS Windows by
Microsoft (among other ports). 

==Requirements==
- SDL 1
- Qt 4
-- "yum install libc SDL libqt4"
